# TV BOX Pro - Landing Page

Uma landing page moderna e responsiva para venda de TV BOX com foco em entretenimento esportivo, especialmente futebol.

## 🚀 Recursos Principais

### Design e Interface
- **Design Moderno**: Interface limpa e profissional com gradientes atraentes
- **Totalmente Responsiva**: Funciona perfeitamente em desktop, tablet e mobile
- **Animações Suaves**: Transições e animações que melhoram a experiência do usuário
- **Background Animado**: Seção hero com elementos de futebol animados

### Funcionalidades
- **Navegação Suave**: Scroll suave entre seções
- **Menu Mobile**: Hambúrguer funcional para dispositivos móveis
- **Seções Interativas**: Cards com efeitos hover e animações ao rolar
- **Modal de Demonstração**: Janela modal para vídeo demonstrativo
- **Integração WhatsApp**: Botões que abrem conversa no WhatsApp
- **Sistema de Notificações**: Feedback visual para ações do usuário

### Conteúdo
- **Hero Section**: Destaque com background de futebol animado
- **Recursos**: 6 benefícios principais do produto
- **Planos**: 3 opções de compra (Básico, Profissional, Premium)
- **Depoimentos**: Prova social com avaliações de clientes
- **CTA Section**: Chamada para ação com benefícios
- **Footer**: Informações de contato e links úteis

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Semântico e acessível
- **CSS3**: Moderno com variáveis CSS, Grid e Flexbox
- **JavaScript ES6+**: Vanilla JS com funcionalidades avançadas
- **Font Awesome**: Ícones profissionais
- **Google Fonts**: Tipografia Inter moderna

## 🎨 Personalização

### Cores
As cores principais podem ser facilmente ajustadas através das variáveis CSS no arquivo `styles.css`:

```css
:root {
    --primary-color: #FF6B35;    /* Laranja principal */
    --secondary-color: #004E89;  /* Azul escuro */
    --accent-color: #00A8E8;     /* Azul claro */
    --dark-color: #1A1A2E;       /* Cor escura */
    --light-color: #F5F5F5;      /* Cor clara */
}
```

### Conteúdo
- **Texto**: Modifique diretamente no HTML
- **Imagens**: Substitua os placeholders pelas imagens reais
- **Contato**: Atualize informações no footer e botões WhatsApp
- **Planos**: Ajuste preços e benefícios conforme necessário

## 🎯 Destaques de Implementação

### Animações de Futebol
- **Bolas Flutuantes**: Elementos ⚽ que se movimentam pelo hero
- **Traves Animadas**: Gols com efeito de pulsação
- **Movimento Contínuo**: Animações suaves e infinitas

### Performance
- **Lazy Loading**: Carregamento otimizado de imagens
- **Intersection Observer**: Animações ativadas apenas quando visíveis
- **CSS Eficiente**: Uso de transform e opacity para melhor performance

### UX/UI
- **Feedback Visual**: Estados hover, loading e notificações
- **Acessibilidade**: Estrutura semântica e navegação por teclado
- **Responsividade**: Breakpoints bem definidos para todos dispositivos
